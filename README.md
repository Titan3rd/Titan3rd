## 🚀 Current Project: LMNT Framework

**LMNT** is a modular Java web framework built on Spring Boot.

Key focus areas:

* Modular architecture for enterprise/business applications
* Reusable backend structure and shared application layout
* React + Vite frontend integration
* Tailwind CSS and shadcn/ui based UI structure
* Authentication, layouts, and reusable framework components
* Product-level extensibility for custom business modules

The goal is to help teams build business web applications faster using a reusable framework foundation.
